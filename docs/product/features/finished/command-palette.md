# Command Palette

The command palette is the fast way to act when you already know your intent.

Open it with `:` or `Ctrl+P`.

What it does today:

- jump to major sections in large maps
- jump to branches by label or id
- jump across outgoing relations and backlinks
- reopen recent places and frequent places
- open saved views
- run built-in recipes
- browse recovery actions like undo, redo, and checkpoints
- preview themes and surface settings
- open help topics

Why it matters:

- it keeps advanced features from turning into a hotkey explosion
- it gives one reliable surface for action, navigation, and recovery
- it works well as a home surface even before you type

Good example queries:

- `browse sections`
- `tasks`
- `product/tasks`
- `#todo`
- `@status:active`
- `backlink`
- `checkpoint`
- `theme`
- `help`
