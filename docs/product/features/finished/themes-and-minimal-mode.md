# Themes And Minimal Mode

`mdmind` now has a real surface system, not just one default theme.

Current built-in themes:

- Workbench
- Paper
- Blueprint
- Calm
- Violet
- Monograph
- Terminal Neon

Related surface features:

- live theme preview from the palette
- per-map UI settings
- minimal mode
- reading mode
- ASCII accents
- attention-guiding motion toggles

Minimal mode is especially useful for experienced users who want:

- less chrome
- more tree width
- quieter overlays
- a more terminal-native feel

Reading mode pairs with that by giving the current node's details a larger calmer pane without changing the active view mode.
