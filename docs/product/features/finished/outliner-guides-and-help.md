# Outliner Guides And Help

`mdmind` now teaches the outliner mental model directly instead of assuming every user thinks “mind map” first.

What ships today:

- built-in help topic for using `mdmind` as an outliner
- built-in help topic for agent workflows
- dedicated repo guides for outliner-first users
- clearer docs for details, reading mode, focused views, and minimal mode as outline features

Why it matters:

- users can approach the product as a calm tree tool before touching the visual mindmap
- writing, planning, research, and meeting-note workflows make sense sooner
- the app is easier to learn without memorizing every advanced feature up front
