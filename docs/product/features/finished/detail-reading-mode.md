# Detail Reading Mode

Reading mode is a surface emphasis layer for detail-heavy work.

It is not another view mode.

The product model is:

- view mode = scope
- minimal mode = chrome density
- reading mode = long-form content emphasis

What ships today:

- a persisted per-map reading mode setting
- palette toggle and preview
- a larger calmer detail reader pane on the right
- placeholder guidance even when the current node has no details yet
- good pairing with Focus Branch, Subtree Only, and minimal mode

Why it matters:

- writing and research maps are easier to read
- node details feel like a real working surface instead of a small preview
- long-form content gets more room without losing the outline
