# Templates And Examples

The project now ships both starter templates and larger example maps.

Templates are for starting work:

- product
- feature
- prompts
- backlog
- writing

Examples are for learning and exploration:

- studio / operating map
- game world map
- novel research and writing map

These matter because they teach the map language in realistic shapes instead of tiny toy snippets.
