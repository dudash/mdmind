# Built-In Help

The integrated help is no longer just a static key list.

Today it gives you:

- searchable help inside the TUI
- user-guide style explanations
- command reference
- practical tips
- grouped topics instead of a flat alphabetical dump
- article scrolling for longer pages

Why it matters:

- users can stay in the terminal while learning features
- advanced features like ids, relations, and node details are easier to discover
- context and search matter more than memorizing every key

This surface is especially useful for:

- learning the query language
- understanding focused views
- remembering relation syntax
- checking safety/recovery behavior
