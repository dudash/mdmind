# Visual Mindmap

The visual mindmap is now a real second lens on the current working set.

What it does today:

- opens from the TUI with `m`
- follows the current view mode
- follows active filters
- renders relation edges when both linked nodes are visible
- exports PNGs
- benefits from the theme system

Why it is useful:

- cluster recognition
- branch shape
- presentation
- seeing cross-branch structure without leaving the tree-first model

The current mindmap is already useful. The future canvas work is about going beyond this overlay, not replacing it.
