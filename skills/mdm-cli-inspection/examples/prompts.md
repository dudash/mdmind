# Example Prompts

Use these prompts to test triggering and output quality for `mdm-cli-inspection`.

## Prompt 1: Validate And Explain

Validate this mdmind file and tell me whether the parser, ids, and relations look clean.
If there are problems, show the smallest mdm command sequence that would help me inspect them.

## Prompt 2: Query Existing Work

Find all blocked launch work owned by jason in this map.
Use the right mdm command and summarize the results clearly.

## Prompt 3: Export A Filtered Subset

Export only the active todo branches from this map as JSON.
Use the narrowest mdm command that gets me a machine-readable result.
